# Lesson 1 — Your First Result

**Goal**: Replace nullable returns and thrown exceptions with an explicit `Result<T>` — one type, all outcomes visible.

**Concepts**: `Result<T>.Ok(value)` · `Result<T>.Fail(error)` · `IsSuccess` · `IsFailure` · `Value` · `Errors`

## Run

```bash
dotnet run
```
